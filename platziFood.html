Descripcion

Restaurante de Comida

En este proyecto se crea un landing page, este sitio web muestra información de un negocio de comida.

Durante la realizacion de este proyecto se utiliza Tailwindcss,el cual es un framework CSS (Hojas de Estilo en Cascada) que permite a los desarrolladores y diseñadores aplicar estilos a los sitios web de una manera ágil y optimizada.

Es una herramienta altamente personalizable. Aunque viene con una configuración predeterminada, se puede sobre-escribir gracias al fichero de configuración tailwind.config.js. Este archivo de configuración permite una fácil personalización de paletas de colores, estilizado, espaciado, temas, etc.
